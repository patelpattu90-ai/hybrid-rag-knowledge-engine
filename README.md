# Hybrid RAG Knowledge Engine

A production-style Retrieval-Augmented Generation (RAG) system for large-scale developer documentation.

## Features

- Programmatic documentation ingestion
- Section-aware parsing
- Token-aware intelligent chunking
- Metadata preservation
- Modular architecture (no LangChain)

## Tech Stack

- FastAPI (API layer - upcoming)
- SentenceTransformers (upcoming)
- FAISS (upcoming)
